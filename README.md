Catering Dashboard
Project Overview

The Catering Dashboard is a web-based application designed to manage and track catering operations efficiently. It provides a centralized platform for administrators and storekeepers to manage inventory, monitor usage, record damaged items, and generate reports. This system streamlines the daily operations of catering departments, ensuring accurate stock tracking and better resource management.

Features
Admin Roles:
Manage users and permissions
View detailed reports on items issued, used, broken, and remaining
Storekeeper Roles:
Update food and utensils inventory
Make store requests
Record used items
Record current stock
Record damaged items
View and search items
Technologies Used
Backend: Java Servlets, JSP
Frontend: HTML, CSS, JSP
